Trivia Master

Le jeu Trivia Master© est un jeu de trivia créé dans le cadre du cours 'Programmation Multiplateforme' du Cégep de l'Outaouais. Pour commencer, on arrive sur le menu du jeu qui est un overlay réalisé grace à CSS et JS. Dans le menu, on a l'option de commencer une partie, de voir la section 'About' ou de voir les instructions du jeu via la section 'How To'. Pour commencer le jeu, on doit choisir une parmi les 24 catégories de question qui sont générées par l'API se trouvant sur le site web Open Trivia Database. L'API me génère un fichier JSON. L'objet comporte la question, trois choix de réponse et la bonne réponse. Une fois la catégorie choisi (un mix de tout les catégories est choisi par défaut) l'utilisateur doit cliquer sur le bouton 'Load Question'.
